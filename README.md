# 全息轨迹
全息轨迹：不止记录跑步、骑行等室外运动轨迹，更能自定义添加室内无轨迹活动，精准记录时长、心率与热量。基于心率-耗氧量线性模型，个性化校准参数，热量计算更精准。轨迹支持 2D/3D 动画回放和录制视频，可添加图文标注，一键导出视频，完整留存每一段运动与生活瞬间。而且对每天所有活动都进行记录，基于一个活动的开始就是前一个活动的结束的理念。
